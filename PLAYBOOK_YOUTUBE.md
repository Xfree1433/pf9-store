## Section C — Per-video metadata refresh (~15 min, batch all 14)

For each video, paste the matching block below into YouTube Studio → Content → click video → Edit.

### Template structure (applies to every video)

- **Title:** keep the original (already SEO'd via `PLAYBOOK_VIDEOS.md`)
- **Description:** new structure below
- **Tags:** copy from per-video lists below
- **Category:** Science & Technology (for ops apps) or Howto & Style (for property/inspection apps)
- **Visibility:** Public
- **Allow embedding:** Yes (default)
- **Chapter timestamps:** YouTube auto-renders them if formatted correctly in the description (every chapter on its own line starting with `0:00`)

### Description template (paste in YouTube; replace `{...}` per video)

```
{One-sentence value prop — same as the video title}

{2–3 sentences expanding on the problem this app solves}

Subscribe at flat-rate pricing — no per-user fees, no contracts, 14-day money-back:
→ https://store.plainspokenfoundrynine.com/for-{manufacturers|landlords}.html

Compare PF9 to per-seat competitors:
→ https://store.plainspokenfoundrynine.com/comparisons/

Run the cost math at your team size or portfolio:
→ https://store.plainspokenfoundrynine.com/tools/per-{user|door}-calculator.html

⏱️ Chapters:
0:00 {Hook — what's the pain}
0:15 {What the app does}
0:45 {Pricing}
1:15 {How to subscribe}

#flatpricing #saas #{manufacturing|propertymanagement} #{appname}
```

## Section F — Maintenance cadence

| Frequency | Action |
|---|---|
| **At publish** | Apply per-video metadata template from Section C |
| **Weekly** | Check YouTube Studio Analytics → Reach → Top traffic sources. Look for queries surfacing your videos that aren't in your tag lists — add them. |
| **Monthly** | Review playlist watch-through rate. If a playlist's drop-off is sharper than others, reorder videos. |
| **Quarterly** | Refresh thumbnails on videos under 100 views — A/B test new thumbnails using YouTube's built-in tool |
